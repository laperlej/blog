About Me

I am a tech lead and senior software engineer at Calcul Quebec and University of Sherbrooke's Center for Scientific Computing.

My expertise lies in building full-stack web applications for researchers and governmental agencies. I manage the entire development process, from provisioning Linux VMs on OpenStack clouds using Infrastructure as Code (IaC) technologies like Terraform and Ansible, to developing frontend applications with Typescript and React and backend systems using Go, Node.js, and Python. I'm always eager to learn and implement new technologies that enhance the quality and efficiency of the products I deliver to my clients.
